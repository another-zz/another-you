# AnotherYou v0.4 - 持久化 AI 社会

> "AI 自主推动世界发展——24/7 永不停止"

## 🎯 核心愿景实现

### ✅ 已实现的关键特性

1. **真正的持久化世界**
   - SQLite 数据库存储所有世界状态
   - 自动保存/加载，永不丢失进度
   - 支持数万 AI 同时在线

2. **AI 终身学习系统**
   - 长期记忆存储（SQLite）
   - 技能学习系统
   - 反思与规划能力

3. **AI 社交关系网络**
   - AI 之间形成关系（朋友/敌人/陌生人）
   - 关系影响行为和互动
   - 关系持久化存储

4. **经济系统基础**
   - 货币系统
   - 资源采集与交易
   - 建造系统

5. **自主推动世界**
   - AI 24/7 自动运行
   - 采集资源、建造、学习、社交
   - 无需玩家干预

## 🚀 快速开始

```bash
# 1. 进入项目目录
cd another-you-v4

# 2. 激活虚拟环境
source venv/bin/activate

# 3. 运行游戏
python main.py
```

## 🎮 操作说明

| 按键 | 功能 |
|------|------|
| WASD/方向键 | 移动 |
| 空格 | 采集资源 |
| B | 建造房屋（需要5木材）|
| I | 打开背包 |
| M | 查看 AI 记忆 |
| R | 查看 AI 关系 |
| TAB | 切换观察的 AI |
| S | 强制保存 |
| H | 帮助 |
| ESC | 退出 |

## 🏗️ 技术架构

```
another-you-v4/
├── main.py              # 游戏主程序
├── data/
│   └── world.db         # SQLite 数据库
└── venv/                # 虚拟环境
```

### 数据库表结构

- **world_state**: 世界状态（时间等）
- **tiles**: 地形数据
- **resources**: 资源分布
- **buildings**: 建筑信息
- **agents**: AI 角色数据
- **memories**: AI 记忆
- **relationships**: AI 关系
- **transactions**: 交易记录
- **events**: 事件日志
