# 指令示例

## 基础指令

```bash
# 移动与探索
"去坐标 (100, 64, 200)"
"找到最近的村庄"
"探索周围 500 格范围"

# 采集与合成
"挖 10 组铁矿"
"砍 5 组橡木"
"合成 64 个火把"

# 建筑
"在我脚下建一个 10x10 的平台"
"建一栋带泳池的现代别墅"
"按照图纸建造一座城堡"
```

## 复杂任务

```bash
# 连锁任务
"去矿洞挖 3 组钻石，然后回家，把钻石放进箱子里"

# 条件任务
"如果天黑了，就回家睡觉；否则继续挖矿"

# 社交任务
"找到玩家 'Steve'，给他 10 个面包"
```

## 个性化指令

```bash
# 学习风格
"记住我喜欢现代简约风格的建筑"
"我通常在晚上挖矿，白天建筑"

# 自主决策
"按照我的风格建一个农场"
"去赚 100 个绿宝石，方法你决定"
```
