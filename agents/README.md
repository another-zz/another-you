# Agent Core - AI分身核心

分层代理架构：

```
┌─────────────────────────────────────┐
│         Planner Agent               │  <- 高层规划
│    (任务分解、目标设定)              │
└─────────────┬───────────────────────┘
              │
┌─────────────▼───────────────────────┐
│         Executor Agent              │  <- 执行层
│    (动作序列、工具调用)              │
└─────────────┬───────────────────────┘
              │
┌─────────────▼───────────────────────┐
│         Perception Agent            │  <- 感知层
│    (环境观察、状态更新)              │
└─────────────────────────────────────┘
```

## 模块说明

- `core/` - 核心Agent实现
- `memory/` - 记忆系统（短期/长期）
- `personality/` - 个性化学习
- `skills/` - 技能库（建筑、挖矿、战斗等）
